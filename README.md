# Installation

Requred '[NodeJS](http://nodejs.org)' installed.

```
$ npm install -g learnyounode
$ npm install -g stream-adventure
$ npm install -g levelmeup
```

# Run

```
$ learnyounode
$ learnyounode list
$ learnyounode run program.js
$ learnyounode verify program.js
```
